Программа ByeDPI для



По-умолчанию, для версии ByeDPI 1.0.1, настройки следующие:



Theme: System

Mode: VPN

DNS: 9.9.9.9

Listen address: 127.0.0.1

Port: 1080

Maximum number of connections: 512

Buffer size: 16384

Default TTL: 0

No domain: Off

Desync only HTTPS and TLS: Off

Desync method: Disorder

Split position: 3

Split at host: Off

TTL of fake packet: 8

SNI of fake packet: www.iana.org

OOB Data: a

Host mixed case: Off

Domain mixed case: Off

Host remove spaces: Off

Split TLS record: Off

TLS record split position: 0

Split TLS record at SNI: Off


Чтобы вернуться к этим настройкам, можно по кнопке в правом верхнем углу выбрать команду "Reset".


В программе есть небольшой глюк: если зайти в настройки, и ничего в них не менять (хотя, даже можно и поменять), то при нажатии кнопки со стрелкой "Назад", произойдет закрытие интерфейса, а не возврат в настройки программы. Чтобы попасть в основное окно программы, надо просто запустить ее снова черз ее ярлык. Настройки будут те, которые были при последней попытке выйти из настроек.
