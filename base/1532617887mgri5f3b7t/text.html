ЗАЛИВКА КАРТ


ПОШАГОВАЯ ИНСТРУКЦИЯ


1. Приобретаем (качаем карту).

2. Закачиваем карту в навигатор по средством проводника (очень важно, чтобы при синхронизации с компьютером навигатор определялся как flash-карта!). Папка для заливки диск:\garmin\файл gmapsupp.img.

Теперь самое главное! Карта работать не будет пока вы ее не разлочите! Как это сделать?

3. Запускаем «garmin unlock generator»

4. В поле «Enter you Unit ID here» вводим ID Вашего навигатора (его можно получить по средствам программы MapSource.

5. Поле «Select Software Product» оставляем как на рисунке (это для навигаторов GARMIN).

6. Далее можно попробовать сгенерировать ключ сразу (НО ЗАЧАСТУЮ ЭТО НЕ РАБОТАЕТ)… И МЫ ИДЕМ ДАЛЬШЕ!!!

7. В поле «Select Map Product» выбираем <custom Mapset>

8. Вписываем в поле «FID (or MapID)» информацию из файла FID.txt (этот файл есть, обычно с каждой картой; если нет, то его можно увидеть в MapSource MapSource, но это уже другая история.

9. Нажимаем «OK», «Generate». Получаем долгожданный ключик!

10. Самое главное! Создаем файл «gmapsupp.unl» (.unl – это расширение), открываем в текстовом редакторе, копируем туда НАШ КЛЮЧИК.

11. Закидываем файл «gmapsupp.unl» в папку с файлом gmapsupp.img

12. ПОЗДРАВЛЯЮ ВЫ РАЗЛОЧИЛИ КАРТУ!
