Чистый образ Raspbian без предустановленный программ и графической оболочки для Raspberry Pi — chisto_rus_raspi.img


Используя загрузчик Raspbian, создал чистый образ без предустановленных программ и графической оболочки.


Сразу установил русскую локализацию в UTF-8 и часовой пояс на Беларусь.


Единственное приложение, которые сразу добавил — это SSH сервер.


Все остальное настроено и установлено "по умолчанию".


Этот образ может пригодится опытным пользователям Linux, которые решили создать систему под себя и не желают расходовать память и дисковое пространство на ненужные им приложения.


Пользователи и пароли:


user userpassword

root rootpassword


Процесс записи образа на флешку описан в посте Raspberry Pi — первый запуск.


Так как образ не содержит никаких стандартных пакетов с программами и нормальная работа с ним возможна только от пользователя root, то рекомендую сразу установить пакет sudo. Для этого в консоли, под пользователем root наберите команду:


apt-get install sudo


Затем откройте в редакторе файл /etc/sudoers


В конец файла добавьте:


user ALL=(ALL) NOPASSWD: ALL


Надеюсь, что на основе этого образа, в сети появятся новые сборки Raspbian под различные нужды пользователей.


Буду признателен, если описание и ссылки на эти образы вы будете оставлять в комментариях к этому посту.

Подключение по SSH


В данных образах для сетевого подключения настроен DHCP-клиент, поэтому чтобы Raspberry Pi получила IP-шник, в сети должен находиться роутер или компьютер с включенным DHCP сервером.
