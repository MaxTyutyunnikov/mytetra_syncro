Так как Laravel использует Composer, то у нас сразу же из коробки есть поддержка фреймворка для тестирования PHPUnit. Устанавливаем этот фреймворк для написания тестов через Composer, и для запуска тестирования просто выполняем команду phpunit из командной строки.

И даже более того, Laravel предоставляет набор хелперов для тестирования самых общих ситуаций в функциональных тестах.

Давайте проверим, что домашняя страница возвращает 200 код ответа.

public function test_contact_page_redirects_user_to_home_page()

{

    $postData = [

        'name' => 'Joe Example',

        'email' => 'email-address',

        'message' => 'I love your website'

    ];

 

    $this->call('POST', '/contact', $postData);

 

    $this->assertRedirectedToRoute('home', null, ['flash_message']);

}
