Если провайдер предоставляет подключение через прямой IP-адрес, то сетевое подключение можно настроить даже через консоль.


Например, если провайдер дает следующие параметры подключения:


IP-адрес: 10.152.210.55

Маска: 255.255.252.0

Шлюз: 12.152.208.1


Тогда создать подключение достаточно просто. Вначале надо выяснить название проводного сетевого интерфейса (на сетевой карте). Делается это командой:


ip addr


Сетевой интерфейс в современных Linux называется примерно так: enp4s0.


Для поключения к сети провайдера через такой интерфейс, можно дать следующие команды:


ip link set dev enp4s0 down

ip link set dev enp4s0 up

ip addr add 10.152.210.55/22 dev enp4s0

ip route add default via 12.152.208.1


Данные команды настроят сеть "на лету". Но это не значит, что эти параметры запомнятся и будут использоваться при загрузке.
