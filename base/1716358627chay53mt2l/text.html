С некоторых пор на сайте фреймверка Qt появился раздел, где выложены все репозитарии старых и новых версий Qt.



Релизы Qt4


Исходники ветки Qt4 доступны по адресу:



https://code.qt.io/cgit/qt/qt.git



Внизу этой страницы даны ссылки для скачивания по различным протоколам - Git, HTTP, HTTPS. Выкачивается полный репозитрий. А чтобы работать с конкретным релизом, надо после выкачивание переключиться на заданную ветку.

Релизы Qt5 и Qt6


Исходники версий Qt5 и Qt6 доступны по адресу:



https://code.qt.io/cgit/qt/qt5.git/
