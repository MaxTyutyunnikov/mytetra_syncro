Просмотр доступных пишущих устройств:


wodim -devices


Стираем перезаписываемый диск:


sudo umount /media/scd0

sudo wodim dev=/dev/scd0 blank=fast


Создадим iso образ директории dir:


1

mkisofs -o image.iso ./dir


Запишем iso образ на диск:


1

wodim -eject -tao speed=16 dev=/dev/scd0 -v -data /home/user/image.iso


Опишу возможные ключи запуска wodim:
-v (-verbose) — подробный вывод информации о процессе записи;
-dao — запись диска в режиме Disc-At-Once, после которого на диск нельзя будет ничего дописать;
speed=16 — скорость записи диска;
dev=/dev/scd0 — путь к пишущему устройству;
image.iso — iso-образ
