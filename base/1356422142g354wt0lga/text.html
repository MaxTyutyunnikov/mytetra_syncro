Таблица создается с помощью команды CREATE TABLE. Пример:


CREATE TABLE artist(

artistid INTEGER PRIMARY KEY AUTOINCREMENT,

artistname TEXT

);


Возможные типы полей:


  • TEXT - текст произвольного размера
  • INTEGER - целое число
  • REAL - дробное число
  • BLOB - бинарные данные
  • TIMESTAMP - метка времени (проверить)

Дополнение типа UNIQUE указывает на то, что значение поля должно быть уникально.


Конструкция INTEGER PRIMARY KEY AUTOINCREMENT является указанием, что поле будет первичным ключем и будет автоинкрементироваться.


Конструкция NOT NULL обозначает, что поле должно быть обязательно заполнено.
