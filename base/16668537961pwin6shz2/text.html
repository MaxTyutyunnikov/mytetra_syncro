Предполагается, что сам Steam уже установлен.


Вот наш план:



  1. Копируем ID игры. У Sims 4 он здесь: https://store.steampowered.com/app/1222670.
  2. Для Windows - Нажимаем Win+R (откроется окошко выполнения команд) и вписываем «steam://install/1222670». В Linux можно попробовать нажать Alt+F2.
  3. Жмем «Okay».
  4. Подтверждаем установку программы.



Все, можно начинать играть.
