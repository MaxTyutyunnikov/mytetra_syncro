В разных версиях дистрибутива Astra Linux были приняты немного различающиеся принципы обозначения текущей версии. Здесь собрана информация о том, как можно однозначно определить версию дистрибутива Astra Linux по специальным файлам, расположенным в каталоге /etc.


Для определения версии и номера обновления можно воспользоваться следующей таблицей:




Astra Linux 1.3 SE

Astra Linux 1.6 SE

Astra Linux 1.7

Файл

/etc/astra_version:

"SE 1.3 (smolensk)"

"SE 1.6 (smolensk)"

"1.7.2"

Файл

/etc/astra_update_version

В Astra 1.3 небыло обновлений и поэтому файла astra_update_version не существует

"Update 10"


Если в системе небыли накатаны обновления, то файла astra_update_version не существует

Файла astra_update_version не существует. Номер обновления кодируется в третьей цифре номера версии в файле astra_version



В кавычках показано содержимое файла. Файл /etc/astra_version для Astra 1.3 и 1.6 являются статическими и не меняются при обновлениях
