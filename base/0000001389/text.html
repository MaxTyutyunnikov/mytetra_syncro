Если есть какой-то файл, и нужно узнать, к какому пакету он относится, можно воспользоваться командой:


apt-file search файл



Внимение! Программа apt-file делает поиск не по индексу, который существует в системе, а по индексу репозитария. И даже если индекс в системе обновлен через команду apt-get update, программа apt-file его использовать не будет. У apt-file хранится свой отдельный индекс, никак не синхронизирующийся с основным системным. Почему так сделано - неясно.


В любом случае, если вы знаете, что файл должен быть, а файл не находится, дайте команду обновления индекса программы apt-file:


apt-file update


И повторите поиск.
