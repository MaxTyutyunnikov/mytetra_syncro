Чтобы разместить на GitHub уже существующий репозитарий, надо сделать две вещи:


1. Создать новый пустой репозитарий на GitHub. Делается это через Web-интерфейс GitHub-а.


2. Находясь в каталоге своего текущего репозитария, дать команды (рекомендуемые самим GitHub) для заливки репозитария:


git remote add origin git@github.com:user/newRepoName.git

git push -u origin master


Первая команда пропишет в настройки локального репозитария удаленный (чтобы он считался как origin). Вторая отправит весь репозитарий на удаленный сервер.


где:


  • user - это имя пользователя на GitHUb
  • newRepoName - это имя созданного пустого репозитария из первого шага.


В результате будет выведено сообщение:


To github.com:user/newRepoName.git

* [new branch]      master -> master

Ветка master отслеживает внешнюю ветку master из origin.
