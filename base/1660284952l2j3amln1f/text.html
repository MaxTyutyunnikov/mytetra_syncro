Официальный источник


Официальный сайт с ГОСТ-ами принадлежит Федеральному агенству по техническому регулированию и метрологии. Их сайт:


https://protect.gost.ru/


Текст документов представлен через постраничную навигацию в виде отрендеренных картинок в высоком качестве в формате jpeg. Для отображения картинки выделен элемент img со статической прозрачной картинкой размером 1x1 pixel. Картинка страницы меняется через CSS-свойство background-image этой однопиксельной картники:



<img src="/i/pixel.gif" class="face" width="1654" height="2339">

...

.face {

background-image: url(image.ashx?page=FCB7B6B2-3F3B-47F6-995D-67FEEAA0718F);

}


Результат:


https://protect.gost.ru/image.ashx?page=FCB7B6B2-3F3B-47F6-995D-67FEEAA0718F



Прочие бесплатные источники


1. Бесплатный на лето 2022 года сайт http://docs.cntd.ru. Называется Электронный фонд правовых и нормативных документов, принадлежит консорциуму Кодекс. На нем можно просто скачивать PDF-файлы.


2. Сайт https://internet-law.ru/gosts - здесь можно посмотреть сканы, но нельзя скачать файл в виде PDF.


3. Сайт https://standartgost.ru/ - обещают что у них можно бесплатно скачать ГОСТы, СанПиНы и другие нормативные документы. На деле при попытке скачивания возникает ошибка: "Error 521 Ray ID: 739724b36e2cb7a9 • 2022-08-12 06:27:07 UTC Web server is down". У некоторых документов кнопки "Скачать" просто нет.


4. Сайт https://gostexpert.ru/
