Иногда возникает необходимость не просто отфильтровать вывод через grep, а показать весь исходный поток символов, выделив в них искомый паттерн.


Для подсветки найденных подстрок можно воспользоваться совокупностью двух опций команды grep.


1. Опция --colour=always


Данная опция включает посветку найденного паттерна. В обычном режиме работы grep, в случае нахождения строки с паттерном, только эта строка будет выведена на экран и в ней будет выделен цветом найденный паттерн.


2. Опция -Cxxx


Данная опция заставляет grep показать заданное количество строк выше и ниже найденной строки. Можно писать -C9999, чтобы гарантирванно показались все строки входного потока. Таким образом, будет выведен весь входной поток, и в нем цветом будут выделены найденные паттерны.


Пример:


openvpn --help | grep --colour=always -C10 mode
