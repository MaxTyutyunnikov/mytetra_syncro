Чтобы удалить файл из индекса, не удаляя его в рабочей директории, можно дать команду:

git rm --cached <file>


эта команда удалит файл из индекса, оставив его при этом в рабочем каталоге.


Такое действие иногда нужно делать после того, как, например, некий тип файла был помещен в .gitignore. Ели файлы этого типа были помещены в репозитарий и стали отслеживаться до того момента, когда они были прописаны в .gitignore, то такие файлы, успевшие попасть в индекс, будут продолжать отслежить
