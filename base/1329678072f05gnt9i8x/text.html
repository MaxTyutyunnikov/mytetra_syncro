Что такое WINEPREFIX?


WINEPREFIX - это переменная окружения, которая содержит путь к каталогу, в котором разворачиваются файлы Windows и файлы программы, которая должна запускаться под Wine. Обычно для каждой Windows-программы создается отдельный "префикс" - то есть отдельный каталог, с которым работает Wine. Это необходимо из-за того, что wine приходится тонко настраивать





WINEPREFIX=$HOME/.wine-wow/ wine "C:\Program Files\WoW\WoW.exe" -opengl
