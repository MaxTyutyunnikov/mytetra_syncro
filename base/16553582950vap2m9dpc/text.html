Для Astra 1.6:


Пакет atftp - блокирует установку обновлений №10 (возможно, и другие выпуски). Для избежания ошибки установки обновления надо данный пакет удалить. От него в системе практически не завязаны другие пакеты, поэтому его удаление не приводит к удалению половины системы.


Пакет libieee1284-3-dev - существует в репозитарии, но по неизвестным причинам не видится через apt-get / apt-cache. Для его установки надо использовать либо aptitude, либо вручную взять файл пакета и установить его через dpkg.
