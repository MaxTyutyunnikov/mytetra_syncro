Для каждого устройства Android получение root - нетривиальная задача. Производители ставят различные препоны народным умельцам в получении root-прав на свой собственный телефон. Здесь описано как рутовать Huawei Honor 6 на примере модели H60-L04.


Внимание! Все действия вы выполняете на свой страх и риск! Только вы несёте ответственность за возможные негативные последствия для вашего устройства. Окирпичивание и прочее нестандартное поведение телефона, которое может произойти после попытки рутования, не является гарантийным случаем. Помните об этом!


Последовательность действий:


  1. Сбор информации о своем телефоне
  2. Активация режима разработчика и отладки по USB
  3. Подготовка к рутованию
  4. Разблокировка загрузчика
  5. Заливка Custom Recovery-прошивки в область Recovery
  6. Установка в телефон программ для рутования
  7. Финальная заливка обычной стоковой прошивки в область Recovery


Для прошивки будут использоваться только пара консольных отладочных программ из Android SDK, поэтому последовательность действий актуальна и для Windows, и для Linux.


Предполагается, что набор утилит ADB уже установлен на компьютер. Эти утилиты берутся с официального сайта:


http://developer.android.com/sdk/index.html


Можно скачать весь Android SDK (тогда будет установлена среда разработки), а можно скачать только SDK Tools Only. Если вы не собираетесь программировать под Андроид, то для рутования достаточно SDK Tools Only. В любом случае, в вашем распоряжении должны оказаться, как минимум, две утилиты под вашу операционную систему: adb и fastboot.


adb - это консольное приложение для компьтера, с помощью которого производится отладка Android устройств, в том числе и эмуляторов. Работает по принципу клиент-сервер. При первом запуске ADB (с любой командой) создается сервер в виде системной службы (демона), которая будет прослушивать все команды, посылаемые на порт 5037. ADB позволяет:


  • Посмотреть какие устройства подключены и могут работать с ADB.
  • Просматривать логи.
  • Копировать файлы с/на аппарат.
  • Устанавливать/Удалять приложения.
  • Удалять (очищать) раздел data.
  • Прошивать (перезаписывать) раздел data.
  • Осуществлять различные скрипты управления.
  • Управлять некоторыми сетевыми параметрами.


fastboot – это приложение для персонального компьютера, работающее через консоль. Данная утилита используется для прошивки файловой системы Android-девайсов. При помощи fastboot можно прошивать на Android-устройство неподписанные образы разделов диска.
