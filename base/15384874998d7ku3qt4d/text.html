Изменение в модуле Qt Bluetooth для Qt.5.11.2:


В BlueZ's DBus LE API портирована реализация Bluetooth Low Energy (минимальная версия для включения - BlueZ 5.42).


Пока не ясно, эта поддержка BLE только для драйвера Bluez, или BLE есть уже для всех драйверов, а в Qt 5.11 добавили поддержку для Bluez. Надо разбираться дальше.
