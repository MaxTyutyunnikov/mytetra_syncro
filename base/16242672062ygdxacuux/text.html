В Debian Linux, в момент, когда подключается Casio CTK-7000 по USB-проводу, появляется следующее устройство:



> dmesg

...

[ 237.204012] usb 1-2: new full-speed USB device number 2 using uhci_hcd

[ 237.397026] usb 1-2: New USB device found, idVendor=07cf, idProduct=6803

[ 237.397028] usb 1-2: New USB device strings: Mfr=1, Product=2, SerialNumber=0

[ 237.397030] usb 1-2: Product: CASIO USB-MIDI

[ 237.397031] usb 1-2: Manufacturer: CASIO



Если вывести список MIDI-секвенсоров в системе, то Casio CTK-7000 будет виден вот так (см. нижнюю строчку):



> aplaymidi -l

Port    Client name                      Port name

14:0    Midi Through                     Midi Through Port-0

24:0    SB Live! 5.1                     EMU10K1 MPU-401 (UART)

25:0    Emu10k1 WaveTable                Emu10k1 Port 0

25:1    Emu10k1 WaveTable                Emu10k1 Port 1

25:2    Emu10k1 WaveTable                Emu10k1 Port 2

25:3    Emu10k1 WaveTable                Emu10k1 Port 3

32:0    CASIO USB-MIDI                   CASIO USB-MIDI MIDI 1
