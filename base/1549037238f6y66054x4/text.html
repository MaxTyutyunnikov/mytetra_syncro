Стандарт BIP


Есть несколько стандартов таких фраз-ключей: BIP 32, BIP 39, BIP 44, и еще какой-то BIP 49, но это не точно. Самый распространенный это BIP 44 (12 слов). BIP 39 это то же самое, только есть возможность выбирать количество слов самостоятельно. Но BIP 39 из 12 слов это то же самое что BIP 44. Если вы забэкапите кошелек именно в этом виде, то вы гарантированно восстановите его почти в любом другом месте через длительное время. Гораздо меньше кошельков поддерживают BIP 39 с 15-тью и более словами. BIP 32 (13 слов) это устаревший формат, его использовать не нужно.



Кошельки


Copay - поддерживает только биткоин, генерирует и восстанавливается по BIP 44/39.

Coinomi - поддерживает очень много валют, восстанавливается по BIP 44/39, но генерирует только BIP 39 из 18-ти слов.

Electrum - поддерживает только биткоин, восстанавливается из BIP44 или BIP 32, но генерирует только BIP 32.

Electrum для DASH и LTC - поддерживают работу только с BIP 32.

Mobile DASH для iOS - поддерживает только DASH, генерирует и восстанавливается по BIP 44. Версия для Android (пока) BIP 44 не поддерживает.
