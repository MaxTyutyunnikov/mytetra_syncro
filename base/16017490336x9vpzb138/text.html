В архиваторе 7z, на самом деле, нет понятия "зашифрованные имена фалов". Тем не менее, имеется возможность создать зашифрованный архив, в котором без ввода пароля не будут видны имена файлов. Делается это с помощью дополнительной опции he. Эта опция отвечает за шифрование заголовка всего архива, и описана так:



Enables or disables archive header encryption.



Команда, с помощью которой создается архив с зашифрованными именами файлов (а по сути, с зашифрованным заголовком), выглядит так:



7z a -r -mhe=on -p<пароль> <имяАрхива>.7z *



Данная команда создаст архив со всеми файлами и подкаталогами текущего каталога.
