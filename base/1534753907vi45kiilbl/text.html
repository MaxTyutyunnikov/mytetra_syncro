Для получения обновлений с удаленного репозитория используется команда:

git pull


Однако, если вы изменили ваши локальные файлы, то команда git pull выдаст ошибку. Если вы уверены, что хотите перезаписать локальные файлы файлами из удаленного репозитория то выполните команды:

git fetch --all

git reset --hard github/master


Вместо github подставьте название нужного удаленного репозитория, (которое ранее было зарегистрировано командой git push -u). Обычно, вместо github пишется origin.


Команда git fetch ... заберет из репозитария все последние изменения. Команда git reset ... установит состояние рабочей директории на последний коммит в ветке master.
