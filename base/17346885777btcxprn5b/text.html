Чтобы у автора проекта с лицензией GPL была возможность распространять проект под различными лицензиями и даже в любой момент поменять лицензию, необходимо, чтобы другие контрибуторы подписали дополнительное соглашение или воспользовались договором-офертой для контрибуторов. В соглашении укзывается, что отправленный в проект код передается безвозмездно, с полной передачей прав на интеллектуальную собственность с возможностью им распоряжаться авторам исходного проекта по своему усмотрению.


Другими словами: когда в GPL-проект контрибутят посторонние люди, то их код далее может распространяться только по GPL-лицензии. Сам же автор, даже если выпустил проект по GPL-лицензии, в праве изменить лицензию новых версий ПО по своему усмотрению, или вообще изначально может распространять свое ПО под разными лицензиями. И чтобы код, закоммиченный в проект сторонними разработчиками, можно было использовать в том же самом проеку
