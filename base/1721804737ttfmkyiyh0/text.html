В экосистеме сайта GitHub имеется специальный репозитарий, в котором хранятся иконки для основных языков программирования и многих известных проектов, нарисованных и опубликованных под свободной лицензией MIT. Иконки хранятся в формате SVG.


Адрес репозитария:



https://github.com/devicons/devicon



Предполагается, что иконки в этом репозитарии, один раз появившись, никуда не перемещаются. Поэтому их можно использовать, как минимум, при оформлении страниц README для собствственных репозитариев любого пользователя.


Чтобы просмотреть возможные иконки в браузере, можно прыгать по подкаталогам и нажимать на *.svg-файлы. Однако для удобства использования был создан специальный родственный сайт-проект:



https://devicon.dev/



Группы иконок представлены в виде табличного списка:





Выбрав группу иконок, справа можно выбрать вариант (стиль) иконки - в светлых тонах, в темных тонах, цветной, черно-белый варианты. После чего можно скопировать ссылку на нужную картику.


Соответствие адресов на devicon.dev и github.com следующее:



https://cdn.jsdelivr.net/gh/devicons/devicon@latest/icons/python/python-original.svg


https://raw.githubusercontent.com/devicons/devicon/master/icons/python/python-original.svg



Использовать этот адрес в README можно следующим образом:



<img src="https://raw.githubusercontent.com/devicons/devicon/master/icons/python/python-original.svg" alt="python" width="128" height="128"/>



Выглядеть такая иконка будет примерно так:





Таким способом можно добавлять в описание проекта необходимые пиктограммы, которые сразу будут бросаться в глаза, обеспечивая тем самым быстрое понимание, какие технологии использует тот или иной проект.
