Команда yes в операционной системе Linux служит для постоянного вывода в стандартный поток строки "yes" или любой другой строки.


Команда yes без параметров будет выдавать строки "yes":



> yes

yes

yes

yes



Команда yes с параметром будет выдавать указанную в параметре строку:



> yes maybe

maybe

maybe

maybe



Команда yes используется обычно в тех случаях, если заранее известно, что какая-то консольная команда во время своей работы будет запрашивать подтверждение на выполнение каких-либо действий. Если команда понимает в качестве подтверждения строку "yes", то использовать утилиту yes можно так:



yes | некая_команда



Если команда понимает в качестве подтверждения символ "Y", то утилиту yes можно использовать так:



yes Y | некая команда
