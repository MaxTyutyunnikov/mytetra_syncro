При работе с сессиями в Laravel можно пользоваться готовой глобальной функцей session().


Вот основные действия, который можно производить с переменными сессии:


// Получение значения переменной

$value = session('key');


// Получение значения переменной с дефолтным значением

// на случай если переменная не установлена

$value = session('key', 'default');


// Устновка значения переменной в сессии

session(['key' => 'value']);


// Удаление переменной из сессии

session()->forget('key');


// Очистить всю сессию (?)

session()->flush();
