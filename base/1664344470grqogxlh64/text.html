Управление профилями Firefoz можно открыть двумя способами.



При запущенном FireFox


В строке адреса необходимо ввести:



about:profiles



и нажать Enter. Откроется скрытая страница О профилях, где будет возможность создать и управлять профилями.



При выключенном FireFox


Необходимо запустить бинарник (exe-шник) firefox с ключем -p. Откроется специальное окно с интерфейсом управления профилями. Это окно будет другим, нежели окно управления внутри FireFox, но основные действия в нем точно такие же.


Зачем сделано два интерфейса управления профилями - не совсем понятно, видимо так исторически сложилось.
