В дистрибутиве Astra Linux имеются следующие секции:


  • contrib
  • main
  • non-free


Однако, сообщается, что еще есть секции:


  • base
  • update
  • extended


Говорят, что пакет build-essential находится в разделе base. Хотя по другим сведениям - в разделе main.


А пакет fail2ban находится в разделе extended.


В любом случе, чтобы были доступны все пакеты, надо в файл /etc/apt/source.list необходимо прописать все необходимые разделы репозитария.
