Бывает так что при запуске виртуальной машины надо указать тип процессора, с которым будет запущена виртуалка. Сделать это можно с помощью опции -cpu, вот так:



qemu-system-x86_64 -cpu IvyBridge-v2 <остальные параметры>



В этой команде будет запущена эмуляция под процессором IvyBridge второй версии.


Надо еще выяснить, какой командой запрашивать перечень возможных типов поддерживаемых процессоров.
