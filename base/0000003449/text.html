Персональный менеджер накопления информации MyTetra обновился до версии 1.28.

Изменения, по сравнению с прошлым крупным релизом 1.25, следущие:

- В системе сборки появилась install-секция

- Сделана русификация

- Текстовые метки стали кликабельными, при клике происходит поиск похожих записей

- Расширена работа с картинками - кумулятивное добавление из файлов, копирование в буфер обмена, редактирование свойств

- Улучшено форматирование в код

- Усилен сброс форматирования (актуально для текстов, вставляемых из браузера)

- Появилось редактирование HTML-исходника

- Новые иконки

- Скрытые возможности (читайте обзор)

- Устранение некритичных багов интерфейса MyTetra и встраиваемого редактора WyEdit

В программе используется только Qt-библиотека, благодаря чему обеспечивается широкая кроссплатформенность. MyTeta v.1.28 доступна для операционных систем Linux, Windows, MacOsX, MeeGo, а сборка из исходников без лишних телодвижений срабатыват под любую в *NIX-операционку, включая FreeBSD.
