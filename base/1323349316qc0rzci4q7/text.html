Универсальный информер, входящий в состав продукта DeskWork позволяет отображать разнообразную справочную информацию: курсы валют, пробки, погоду, биржевые индексы и пр. Модуль использует информацию из открытых источников в Интернете: специализированные информ-сайты с коллекциями из сотен готовых инфо-блоков

Адресная книга - Список сотрудников (телефонный справочник, адресная книга) с возможностью сортировки по любому из полей и поиска сотрудника по полям имени и фамилии особенно удобен в использовании, так как обращается за исходной информацией в каталог Active Directory, а также позволяет импортировать данные о пользователях из xml-файлов. С помощью импорта из xml-файлов возможна интеграция с другими системами, например, получение информации о сотрудниках из существующих программ управления персоналом и учета кадров.
