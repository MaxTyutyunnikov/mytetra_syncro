При создании очередного проекта в Git, следует проверить, какой пользователь задан для проекта по-умолчанию. Обычно берется глобальная настройка Git.


Посмотреть, какой пользователь задан глобально в Git, можно с помощью команды:



> git config --list

...

user.name=Vasya

user.email=vasya@mail.ru

...



Если необходимо изменить Git-пользователя глобально, используются следующие команды:



> git config --global user.name "Petya"

> git config --global user.email petya_hlopushkin@example.com



Данные действия надо не забывать делать вначале при настроке проекта на рабочей станции разработчика.
