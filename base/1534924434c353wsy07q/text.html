Если нужно открывать окно какой-либо программы в одном месте экрана, то это можно настроить так.


В начале надо запустить нужную программу (например, KWrite), и выставить его окно так, как оно должно находиться. Далее надо в консоли запустить программу xprop. Появится крестик, которым нужно щелкнуть на заголовке окна. Перед выбором окна можно нажимать Alt-Tab чтобы выбрать нужное окно. После клика в консоль выведется информация об окне программы. Нужны будут следующие данные:


$ xprop

_OB_APP_TYPE(UTF8_STRING) = "normal"

_OB_APP_CLASS(UTF8_STRING) = "kwrite"

_OB_APP_NAME(UTF8_STRING) = "kwrite"

WM_WINDOW_ROLE(STRING) = "MainWindow#1"


WM_NORMAL_HINTS(WM_SIZE_HINTS):

user specified size: 1053 by 841


Далее надо открыть файл /home/xi/.config/openbox/lxde-rc.xml, и в секцию </applications> добавить такой код:


<application name="kwrite" class="kwrite" type="normal">

<position>

<x>384</x>

<y>0</y>

</position>

<size>

<width>1056</width>

<height>842</height>

</size>

</application>


Для тега application еще можно задать атрибут role="MainWindow#1". Это нужно будет сделать в случае, если другие окна KWrite (например, окно настройки KWrite), будут открываться на весь размер основного окна. То есть, опциями в теге application точно настривается именно то окно, размер и положение которого надо выставить автоматически.


В строке "user specified size" программа xprop показывает размеры окна без учета границ. Поэтому ширину и высоту надо немного подкорректировать. А координаты окна xprop вообще не показывает, их надо задать вручную.


После внесения настроек, надо перечитать конфигурацию. Делается это командой:


$ openbox --reconfigure


После чего можно запустить программу, и ее окно должно открыться в нужном месте с нужным размером.
