Если нужно открывать окно какой-либо программы в одном месте экрана, то это можно настроить так.


В начале надо запустить нужную программу, и выставить его так, как оно должно находиться. Далее надо в консоли запустить программу xprop. Появится крестик, которым нужно щелкнуть на заголовке окна. Перед выбором окна, можно переключать по Alt-Tab чтобы выбрать нужное окно. После клика в консоль выведется информация об окне программы. Нужны будут следующие данные:


$ xprop

_OB_APP_TYPE(UTF8_STRING) = "normal"

_OB_APP_CLASS(UTF8_STRING) = "kwrite"

_OB_APP_NAME(UTF8_STRING) = "kwrite"

WM_WINDOW_ROLE(STRING) = "MainWindow#1"


WM_NORMAL_HINTS(WM_SIZE_HINTS):

user specified size: 1053 by 841
