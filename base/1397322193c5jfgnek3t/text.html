Если при открытии окна Network Management видна ошибка:


Kde4 network management is not running. Please start it.


Это значит, что linux-сервис с именем network-manager не запущен.


Запустить его можно командой:


service network-manager start


После выполнения этой команды ошибка должна исчезнуть. Однако, это не значит, что network-manager прописался в автозагрузку. Вышеприведенная команда только разово запустит network-manager.


Чтобы прописать запуск network-manager в автозагрузку, нужно под рутом вызвать команду:


sysv-rc-conf


И для сервиса network-m$ установить запуск на уровнях 2-5. Строка "network-m$" - это сокращение от "network-manager".


service 1 2 3 4 5 0 6 S ----------------------------------------------------------------------------
