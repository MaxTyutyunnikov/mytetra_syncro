В силу внутренних особенностей архитектуры фреймверка Laravel, традиционные IDE для работы с PHP-кодом не подходят.


Например, в NetBeans нет плагина, который поддерживал бы синтаксис blade-файлов. А так же нет банальной возможности перехода к файлу вида, так как он пишется в виде строкового параметра функции.


Конечно, такие возможности есть в PHPStorm, но он платный. А сущетсвует ли редактор кода, который был бы бесплатным, с OpenSource лицензией, и в котором были бы все необходимые возможности для работы с инфраструкторурой Laravel? Да еще чтобы он был кроссплатформенным и работал в Linux/Windows/MacOs? Оказывается, такой редактор есть. Это Visual Studio Code от компании Microsoft.


Visual Studio Code позиционируется как бесплатный легковесный редактор, имеющий MIT-лицензию. Но благодаря системе плагинов, он расширяется очень неплохо. По-сути, у него один недостаток - этот редактор жрет память как не в себя, так как основан на Electron.


Посмотреть, какие расширения для работы с Laravel доступны для этого редактора, можно на следующей странице:



https://marketplace.visualstudio.com/search?term=laravel&target=VSCode&category=All%20categories&sortBy=Relevance



В частности, там есть такие плагины:


  • Laravel Blade Snippets - Laravel blade snippets and syntax highlight support
  • Laravel goto view - Quick jump to view
  • laravel-goto-controller - Alt + click to navigate from a route to a respective controller file


Из свободного и бесплатного, для работы с Laravel, лучшего решения на текущий момент просто нет.
