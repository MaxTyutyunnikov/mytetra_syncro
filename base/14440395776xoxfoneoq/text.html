Общеизвестно, что VirtualBox, после покупки его компанией Oracle в 2010 году, стал выходить в виде крайне нестабильных релизов. Глючные релизы выпускаются под разные операционки. Какая-то версия VirtualBox не работает или жутко глючит в Linux, какая-то в Windows.


Здесь я пытаюсь собрать сведения о хороших релизаци VirtualBox.


1. Более-менее стабильные версии VirtualBox выпускались в третьей версии. Если нет необходимости в новых фичах, проще всего использовать версии 3.x.x


2. Информация с Хабара: ещё в апреле 2015 мне пришлось откатиться к VB 4.3.12 на Windows 7 из-за бага "can't start VM 0x80004005".


3. Положительные отзывы о версии 4.3.20 под Linux есть на Linux.org.ru


4. VirtualBox 5.0.6 - нормально работает под Linux, советуют переходить с ветки 4.3.x на Opennet.ru. Однако эта же версия на Windows 7 вываливает всю систему (хост-систему!) в BSOD, при попытке запуска ReactOs.
