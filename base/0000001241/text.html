Создать ссылку на файл можно так:


ln -s file.txt linkfilename.txt


где


  • -s - опция, указывающая создавать мягкую (символическую) ссылку
  • file.txt - файл, который уже существует
  • linkfilename.txt - имя создаваемого файла линка


Вместо имени файла можно указать имя директории:


ln -s /home/user/collect/mydir /home/user/mydir
