Чтобы посмотреть историю изменения одного файла, надо дать команду:


git log -p имя_файла


Будут выведены все изменения в виде diff синтаксиса


Еще можно воспользоваться командой:


git blame имя_файла


Данная команда выведет построчно файл, причем для каждой строки будет указан коммит, автор, и время фиксации. Иногда такой вывод очень удобен, например, чтобы узнать, кто конкретно внес и
