Чтобы посмотреть историю изменения одного файла, надо дать команду:



git log -p имя_файла



Будут выведены все изменения в виде diff синтаксиса


Если нужно посмотреть не историю изменений файла, а историю того, кто изменял существующие строки файла, то можно воспользоваться командой:



git blame имя_файла



Данная команда выведет построчно файл, причем для каждой строки будет указан коммит, автор, и время фиксации. Именно такой вывод очень удобен чтобы быстро узнать, кто конкретно и когда внес изменения в интересуемые строки файла. Здесь нужно понимать, что будут отображаться только существующие на данный момент строки, а не добавляемые/изменяемые/удаляемые строки файла.
