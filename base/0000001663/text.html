Если удалить с помощью команды svn delete каталог, то произойдет следующее.

Все файлы и подкаталоги внутри каталога будут удалены. А сам удаляемый каталог останется на диске. Внутри этого каталога будет находиться только директория .svn. В этой директории в специальном файле будет прописано удаление данного каталога.

Окончательное удаление каталога с диска произойдет после обращения к серверу через команду svn commit.
