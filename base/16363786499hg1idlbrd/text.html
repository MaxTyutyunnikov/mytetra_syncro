По-умолчанию файловая система /system, доступная через ADB, примонтирована в режиме только для чтения.


Исправить данную ситуацию можно с помощью команды перемонтирования. Выглядит эта команда так:



mount -o rw,remount /system



Вернуть обратно можно так:



mount -o ro,remount /system



Кроме каталога /system в режиме read only по-умолчанию примонтированы и другие каталоги. Их так же можно перемонтировать в режим, позволяющий запись на диск.


Узнать все примонтированные устройства/каталоги можно с помощью команды:



mount



Для выполнения самого перемонтирования, возможно, потребуются права рута, которые можно получить через следующую команду (если, конечно, телефон рутован):



su -



Естественно, все вышеприведенные команды нужно выполнять в ADB-шелле, который вызывается на компьютере, к которому подключен сотовый телефон. Сам ADB-шелл запускается командой:



adb shell



По-умолчанию вход в данный шелл происходит под именем пользователя shell, что видно в приглашении ввода.
