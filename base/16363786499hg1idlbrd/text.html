По-умолчанию файловая система /system, доступная через ADB, примонтирована в режиме только для чтения.


Исправить данную ситуацию можно с помощью команды перемонтирования. Выглядит эта команда так:



mount -o rw,remount /system



Вернуть обратно можно так:



mount -o ro,remount /system
