На панельных компьютерах Орион ПК 103 (ПК-Э-103-02) в дистрибутиве Astra Linux 1.6 из-коробки не работает видеодрайвер. Следующие действия помогут исправить эту проблему.


1. На этапе загрузки системы выбрать режим recovery mode.


2. Переключиться в режим терминала: Alt+F1.


3. Авторизоваться под пользователем-администратором, из-под которого возможно переключение на пользоваиеля root. Авторизация в режиме level: 0.


4. Установить пароль для пользователя root:


sudo passwd root


5. Открыть термирнал, сменить пользователя на root:


su -


6. Отредактировать файл загрузчика grub:


mcedit /etc/default/grub


Добавить строку:


GRUB_CMDLINE_LINUX_DEFAULT="nomodeset"

Выполнить команду:


update-grub2


7. Создать файл /etc/X11/xorg.conf.d/20-intel.conf со следующим содержимым:


Section "Device"

Identifier "Intel GMA3600"

Driver "modesetting"

Option "AccelMethod" "none"

EndeSection


8. Перезагрузить систему:


reboot


9. При необходимости сменить режим планшета в меню "Тип сессии" на этапе входа в систему.
