В сети имеется интересный сайт, на котором можно найти полную документацию (и даже гораздо больше) по системе контроля версий Git. Сайт является генератором man-страниц по всем возможностям git:



https://git-man-page-generator.lokaltog.net



Верстка страниц сделана хорошо, элементы кода выделяются моноширинным шрифтом, цветовая палитра не яркая (но, к сожалению, не регулируемая). Чтение текста документации комфортное.


Пример страницы на команду git-trip-hash:

Авторы книг по Git, обычно, в качестве первоисточника документальной технической информации, активно используют данный сайт.
