MyGpsFiles


Есть хороший сервис:


http://www.mygpsfiles.com/app/


Если трек сделан с временными метками, то с помощью движка можно перемещаться по треку согласно времени.


В качестве подложки используется OpenStreetMap.


Выглядит его интерфейс так:





Регистрация не требуется, сервис свободный и бесплатный.



GpxEditor


Еще есть сервис, который показывает GPX-трек с подложками GoogleMaps и OpenStreetMap


http://www.gpxeditor.co.uk/


Чтобы загрузить GPX-трек, надо нажать на кнопку с открывающейся папкой. Указывается локальный файл, и он будет отображен на карте.





Регистрации так же не требуется.
