Чтобы в веб-интерфейсе роутера MikroTik hAP Lite сохранить настроенную в роутере конфигурацию, надо перейти в интерфейс WebFig:





В меню надо выбрать пункт Files. Откроется окно сохраненных на роутере файлов. В нем, для создания резервной копии настроек роутера, надо нажать кнопку Backup.





Откроется окно создания резервной копии:





Почему поле ввода имени файла и пароля представляет собой раскрывающийся список, известно только высококлассным специалистам по юзабилити компании-производителя роутера MikroTik. Чтобы появилось поле ввода, надо нажать на стрелку вниз:





При необходимости, можно поставить галку Don't Encrypt, тогда полученный файл не будет шифроваться. После нажатия Backup, в списке файлов появится новый файл с сохраненной конфигурацией.


Далее, чтобы сохранить файл конфигурации на компьютер, нажимается кнопка Download возле соответствующего файла в списке.


Если нужно загрузить на роутер файл конфигурации с компьютера, необходимо в поле Upload (находится сверху над списком) нажать кнопку выбора файла. После загрузки файл появится в списке файлов.


Теперь остался вопрос: а как восстановить конфигурацию из файла? Никакой кнопки, которая бы запустила действие восстановления конфигурации из файла, в интерфейсе не наблюдается. Как быть?


Для этого надо просто нажать на строку с файлом. Не на кнопку [-] и не на кнопку [Download], а просто тыкнуть на имя файла, из которого необходимо восстановить конфигурацию. Откроется окошко восстановления:





В этом окне следует нажать кнопку Restore и конфигурация роутера будет восстановлена.
