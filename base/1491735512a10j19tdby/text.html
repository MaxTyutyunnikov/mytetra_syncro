Для unix вводим следующие строки в консоли:


curl -sS https://getcomposer.org/installer | php

mv composer.phar /usr/local/bin/composer


Для проверки версии установленного composer надо дать команду:


composer --version


или


php composer --version


Все необходимые пакеты, которые должен установить composer, прописываются в файле composer.json. Для того чтобы создать его, необходимо ввести следующую команду в консоли, находясь внутри папки с вашим проектом:


composer init


Если php-скрипты не выполняются автоматически, то во всех командах слово composer заменяется на php composer. То есть так:


php composer init


Дальше вы указываете «композитору» как назвать ваш проект и так далее, а так же находите пакет который хотите установить, выбираете его, а потом смотрите необходимую его версию на сайте https://packagist.org.

После чего в папке вашего проекта появится файл composer.json, со всеми описанными зависимостями. Теперь осталось их только установить. И делается это следующей командой:


composer install


Все. После этого можете с улыбкой наблюдать как в вашем проекте появилось все что вы хотели скачать.
