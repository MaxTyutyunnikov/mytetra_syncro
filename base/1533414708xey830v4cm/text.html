Перемещение по истории посещенных файлов


Самый востребованный инструмент разработчика - это перемещение по истории. Обычно, программисту приходится работать всего с несколькими файлами, быстро переключаясь между ними.


Переключаться щелчком по файлу в дереве проекта - это слишком долго и неудобно, так как проекты обычно большие. Поэтому было придумано окно открытых файлов. Но в окне открытых файлов быстро скапливается большое количество файлов, которые программист редактировал при той или иной работе. Поэтому список редактируемых файлов нужно либо постоянно чистить, либо придумывать что-то еще. Это "что-то еще" и есть навигация по истории просмотренных/отредактированных файлов.


В QtCreator навигаци по истории делается с помощью комбинации клавиш:


Ctrl + TAB


Будет открыто "всплывающее" окно со списком посещенных файлов. Вверху будут последние посещенные файлы. Не отпуская клавишу Ctrl можно нажимать клавиши со стрелками Вверх/Вниз, и выбрать нужный файл.


Есть одна тонкость: файл будет выбран либо при отпускании Ctrl, либо при нажатии Enter (не отпуская Ctrl). Часто "по инерции" нажимается Enter после отпускания Ctrl, и тогда в файле будет добавлена новая строка. Надо за этим следить.



Перемещение по закладкам


Чтобы поставить закладку на строку, используется комбинация:


Ctrl + M


Где буква M обозначает Memory (типа запомнить в память место где ты находишься).


В стандартной конфигурации перемещение по закладкам в пределах всего проекта настроено через клавиши:


  • Ctrl + . - следующая закладка
  • Ctrl + , - предыдущая закладка


Так же можно вызвать вкладку списка закладок через комбинацию:


Alt + M


При первом запуске данной вкладки она может содержать странный список файлов, совершенно не похожий на перечень закладок. Это баг среды QtCreator. Надо эту вкладку закрыть кнопкой сворачивания вкладки, и снова открыть с помощью вышеуказанной комбинации. После таких действий список закладок будет отображаться корректно.


Настройки горячих клавиш для работы с закладками можно сделать в меню Инструменты - Параметры - Среда - Клавиатура.
