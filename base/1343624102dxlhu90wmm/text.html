Использование клавиш X, C, V для действий Cut, Copy, Paste


Эта прохладная история о соглашении использовать клавиши X, C, V для действий Cut, Copy, Paste дошла до нас из компании Apple. Если посмотреть на клавиатуру, то расположение клавиш в левой части клавиатуры будет следующим:


Q W

A S D

Z X C V


Компания Apple в 1980-х годах сделала концепцию вырезать/скопировать/вставить популярной для всех пользователей своих компьютеров. А так же она сформировала концепцию клавиатурных сокращений, называемых в русском языке горячими клавишами, а в английском - шорткатами. Вышеуказанные буквы были выбраны потому, что они во-первых находятся в левой части клавиатуры, а во-вторых, под ними (внизу) находится клавиша ⌘ Cmd. Таким образом появились следующие сочетания:


  • Quit - Cmd + Q,
  • Close - Cmd + W,
  • Select All - Cmd + A,
  • Save - Cmd + S,
  • Duplicate - Cmd + D,
  • Undo - Cmd + Z


Исключением из этого принципа стали кнопки Cmd+O для действия Open (открытие файла) и Cmd+P для действия Print (печать), которые находятся в правой части клавиатуры.


В настоящее время все эти клавиатурные сочетания стали, де-факто, стандартом для всех приложений в операционных системах Windows, Mac, Linux, за исключением Z для отмены (Undo) (видимо, имеется в виду, что используется Ctrl+U) и D для дублирования (Duplicate) (видимо, имеется в виду, что Ctrl+D используется для удаления строк в редакторах).


Вышеприведенное объяснение сумбурное, и не отвечает на вопрос, почему таки используются клавиши X,C, V. Ну вот такие англоговорящие люди рассказывают такие непоследовательные истории.



Оригинал на английском


The X C V for Cut Copy Paste


Another history is the convention of X C V keys for Cut Copy Paste. This came from Apple.


Q W

A S D

Z X C V


Apple computer, in the 1980s, made the undo, cut, copy, paste concepts popular to the masses, and in general the computer keyboard shortcuts concept. These keys are chosen because they are all adjacent and on the left side of the keyboard, where under them there's the ⌘ Cmd key. Also in this set are Quit Q, Close W, Select All A, Save S, Duplicate D, and Undo Z. The only exceptions are Open O and Print P on the right side of keyboard.


All these keys have become de facto standard on all applications on {Windows, Mac, Linux} today, except the Z for undo and D for Duplicate.
