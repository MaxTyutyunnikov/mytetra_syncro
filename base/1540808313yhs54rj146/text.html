Если известно имя шрифта, то можно попробовать найти его файл. Перед тем как искать, нужно знать, что в Linux обычно установлено несколько видов шрифтов, которые хранятся в различных форматах. Бывает несколько форматов: Type1 (расширение *.pfb), OpenType (расширение *.otf), True Type Font (расширение *.ttf) и другие.

Традиционное размещение файлов со шрифтами:


/usr/share/fonts/


Например, есть шрифт Free Helvetian Condensed. Найти его файл можно командой:


fc-match -v Condensed | grep file


а если эта команда ничего вразумительного не найдет, то скорее всего поможет команда:


fc-list | grep -i "Condensed"


А так же есть утилита fc-scan, которая тоже может помочь при поиске файла шрифта.


При поиске файла шрифта лучше указывать не его полное имя с пробелами, а часть имени, потому что имя шрифта формируется зачастую из соответствующих полей с теми или иными свойствами.
