Для включения WaveTable-синтеза на карточке SBLive 5.1, можно воспользоваться следующими командами. Команды следует выполнять от рута:



service timidity stop

modprobe snd-emu10k1-synth

asfxload /usr/share/sounds/sf2/8MBGMSFX.SF2



Остановка сервиса timidity нужна для того, чтобы программы-секвенсоры MIDI не подхватывали автоматически его, а работали через аппаратный Wave Table.


Для SBLive 5.1 подходят банки звуков инструментов размером 8Мб. Взять такой банк можо на официальном сайте разработчиков звуковой подсистемы ALSA:



http://www.alsa-project.org/~james/sound-fonts/8MBGMSFX.SF2



UPD: сохраненная копия этого файла приложена к данной записи.
