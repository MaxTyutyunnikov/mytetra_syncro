Пояснение


Фабрика объектов - это такой класс, который способен порождать объекты разных классов. Главное, чтобы эти классы имели один общий базовый класс.


Предполагается, что вначале в фабрике "регистрируются классы", которые может создавать фабрика, после чего фабрика может создавать объекты заданного класса.


Фраза "регистрируются классы" не просто так взята в кавычки. Естественно, "добавлять классы" в фабрику и "хранить список классов" в фабрике язык C++ не позволяет.


Вместо этого, для каждого класса создается объект creator (причем все классы creator-ов созданы на основе одного родительского класса), и именно он добавляется в фабрику через метод add(). И при создании объекта нужного класса через метод create(), вызывается соответствующий creator.
