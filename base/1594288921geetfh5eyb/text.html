Устанавливать надо из PPA, пакеты подходят для Debian 9/10.


Вначале надо в файл /etc/apt/sources.list прописать строку репозитария:



# Ansible Launchpad repository

deb http://ppa.launchpad.net/ansible/ansible/ubuntu trusty main



Затем можно запустить обновление пакетов через команду apt-get update. В результате будет получена ошибка:



> apt-get update

...

Ошб:4 http://ppa.launchpad.net/ansible/ansible/ubuntu trusty InRelease

Следующие подписи не могут быть проверены, так как недоступен открытый ключ: NO_PUBKEY 93C4A3FD7BB9C367

...

Чтение списков пакетов… Готово

W: Ошибка GPG: http://ppa.launchpad.net/ansible/ansible/ubuntu trusty InRelease: Следующие подписи не могут быть проверены, так как не

доступен открытый ключ: NO_PUBKEY 93C4A3FD7BB9C367

E: Репозиторий «http://ppa.launchpad.net/ansible/ansible/ubuntu trusty InRelease» не подписан.

N: Обновление из этого репозитория нельзя выполнить безопасным способом, поэтому по умолчанию он отключён.



Чтобы добавить публичный ключ данного репозитария, надо дать команды:



gpg --keyserver keyserver.ubuntu.com --recv 93C4A3FD7BB9C367

gpg --export --armor 93C4A3FD7BB9C367 | sudo apt-key add -



После чего надо заново запустить apt-get update, эта команда должна сработать без ошибок.



Далее можно запустить Synaptic, и в свойствах пакета ansible, на закладке "Версии" можно будет увидеть две версии этого пакета - из репозитария Debian и из репозитария PPA. Для установки нужной версии, необходимо выделить пакет ansible, затем в меню нажать "Пакет" - "Указать версию...". После выбора нужной версии, следует нажать кнопку "Применить", и необходимая версия будет установлена в систему.


Например, для Debian 9, вместо версии 2.2.0.1-2 можно установить 2.9.7-1ppa. А для Debian 10 вместо версии 2.7.7 можно установить 2.9.10-1ppa.
