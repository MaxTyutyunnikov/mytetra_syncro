Unsplash


Сайт: https://unsplash.com/


Тип лицензии: CC0.

Количество изображений: более 2 млн.

Фотосток с огромным выбором изображений на разные темы. Каждое из них можно скачать в разных разрешениях без регистрации.

Перед скачиванием изображения можно посмотреть количество его просмотров и загрузок. Также бывают доступны данные о фотокамере, на которую сделали снимок.

При просмотре фотографий сток подбирает похожие снимки, а также похожие теги и коллекции.

Выдачу можно отфильтровать по цвету и ориентации фото, отсортировать по новизне или релевантности.



StockSnap


Сайт: https://stocksnap.io


Тип лицензии: Лицензия CC0.

Количество изображений: более 10 000.

Все работы можно скачать без регистрации и в высоком качестве.

Есть поиск по изображениям.

В каталоге доступны два основных раздела:

1. Категории:

2. Все фото:

Тут находятся все изображения, представленные на ресурсе. По умолчанию они отсортированы по дате: сначала отображаются недавно добавленные снимки.

Также доступна сортировка по другим параметрам:

Trending – по популярности;

Views – по числу просмотров;

Downloads – по числу загрузок;

Favorites – по числу лайков (добавлений в избранное).



Pexels


Сайт: https://www.pexels.com

Тип лицензии: CC0.

Количество изображений: более 30 000.

Популярный фотобанк с качественными изображениями разных тематик и стилей.

Можно скачивать изображения в нескольких стандартных или кастомных разрешениях. Для загрузки не требуется регистрация.

Имеются тематические коллекции и коллекции лучших изображений, группировка по более чем 150 категориям.

Кроме фотографий на сайте есть фоновые (в т. ч. анимированные) изображения. Если нужны изображения в конкретной цветовой гамме, можно использовать поиск фото по цвету.



Flickr


Сайт: https://flickr.com/


Тип лицензии: все виды лицензии, в том числе CC0.

Количество изображений: более 1 млрд.

Flickr является частью Yahoo!. Его каталог превышает миллиарды фотографий. Изображения загружают простые пользователи, поэтому иногда встречаются работы не в самом высоком качестве. У каждого изображения на Flickr есть собственная лицензия, поэтому нужно ее предварительно проверять. Для скачивания нужна регистрация.
