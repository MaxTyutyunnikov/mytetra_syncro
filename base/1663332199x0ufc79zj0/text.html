Royalty-Free

Эта лицензия применяется для платных изображений.



Тип лицензии

Что можно делать с изображениями

Royalty-Free

Пользоваться изображением без ограничений, но не больше 500 000 раз на разных носителях.

Extended Royalty-Free

Пользоваться изображением без ограничений по количеству использований.



Rights Managed

По этой лицензии изображение можно использовать только один раз после приобретения. Если вам нужно воспользоваться им несколько раз, то придется покупать его столько раз, сколько нужно.
