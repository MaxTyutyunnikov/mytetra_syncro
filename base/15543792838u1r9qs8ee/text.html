В проектах Laravel часто используется сборка и компиляция ресурсов: css-файл объекдиняются в один большой файл, js-файлы тоже минимизируются и объединяются. Сборка производится через подсистему Webpack.


Поэтому после изменений, внесенных в ресурсы, надо пересобрать файлы, которые будут использоваться на сайте.


Если сайт находится в разработке, для пересборки файлов ресурсов можно запустить команду:


npm run dev


А для сборки в продакшен можно запустить так:


npm run production
