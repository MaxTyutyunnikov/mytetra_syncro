Для отключения JavaScript в браузере FireFox, необходимо произвести следующие действия:


  1. Запустите браузер и откройте новое окно.
  2. В адресной строке напишите «about:config» и нажмите «Enter».
  3. Появится предупреждение, в котором нужно будет принять условия.
  4. Напишите в строке поиска «javascript.enabled».
  5. После этого будет выведена строка настройки JS. В ней выберите колонку «Value», чтобы включить JavaScript в Firefox, и замените значение на «True».
  6. Обновите страницу с помощью клавиши «F5».
