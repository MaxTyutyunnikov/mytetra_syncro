Если попытаться установить программу из deb-файла, то даже если в этом пакете будут прописаны зависимости, они не будут устанавливаться из прописанного в системе репозитария. Для решения этой проблемы можно выполнить следующие действия.


Копируем нужный для установки deb-файл в следующий каталог:



/var/cache/apt/archives/



И даем команду установки будто это обычный пакет:



apt-get install "имя deb файла без расширения"



или



aptitude install "имя deb файла без расширения"



Важно обратить внимание, что имя файла пакета указывается в этих командах без расширения.


В процессе установки apt/aptitude сам предложит установить все необходимые пакеты.


Программе apt-get можно добавить опцию -y, благодаря которой все зависимости будут ставиться автоматически, на спрашивая подтверждения на установку.
