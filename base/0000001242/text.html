Если checkinstall по какой-то причине работает неправильно, можно пакет создавать вручную.


Для это можно сделать следующее.


1. Создать новую директорию, например /tmp/package


2. Находясь в каталоге с исходниками, выполнить команду



make install DESTDIR=/tmp/package



3. Зайти в каталог /tmp/package и выполнить команду



dpkg-deb -b package_name.deb



В результате будет собран deb-пакет, который можно установить через пакетную систему Debian-based дистрибутива.
