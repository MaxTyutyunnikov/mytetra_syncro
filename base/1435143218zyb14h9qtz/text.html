Страница входа конструироуется с помощью asp-скрипта logon.aspx, который обычно находится по следующему пути:


C:\Program Files\Microsoft\Exchange Server\V14\ClientAccess\Owa\auth\logon.aspx


Его можно произвольно менять. Если нужно добавить надписи на русском языке, следует знать, что кодировка этого файла - cp1251. Далее данный файл, при выдаче через web-сервер, переконвертируется в кодировку UTF-8.
