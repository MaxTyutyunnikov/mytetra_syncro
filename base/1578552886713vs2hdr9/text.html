Файлы в формате DjVu - это, обычно, отсканированные документы, книги, учебники, научная литература и так далее, записанные с применением особого алгоритма сжатия. Данный алгоритм хорошо работает именно со сканированными печатными изображениями.

Для конвертирования DjVu в PDF можно использовать пакет djvulibre-bin. Установка пакета:


sudo apt-get install djvulibre-bin


Теперь займёмся конвертацией. Допустим в директории лежит файл Книга.djvu и его нужно сконвертировать в pdf. Не проблема, нужно воспользоваться такой командой:


ddjvu -format=pdf Книга.{djvu,pdf}


Да, формат команды очень необычный. В любом случае, после завершения команды, в текущем каталоге появится файл Книга.pdf.
