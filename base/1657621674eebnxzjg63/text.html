Вопрос: нам нужно, чтобы при старте рабочего стола у всех пользователей был включен NumLock. Для этого в файле /etc/X11/fly-dmrc мы раскомментировали опцию:



NumLock=On



Однако при входе в систему NumLock все равно не включен. Как для всех пользователей включить NumLock?


Ответ: для этого в домашнем каталоге пользователя предлагается с помощью текстового редактора отредактировать конфигурационный файл ~/.fly/theme/current.themerc и для новых пользователей файл /usr/share/fly-wm/theme/default.themerc, где строку:



numLockOn=false



следует привести к виду:



numLockOn=true



Еще необходимо обратить внимание на параметр numLockIndicator.


Если во все домашние каталоги не прописывать, то можно попробовать сделать файл



/usr/share/fly-wm/theme.master/themerc



где в секции [Variables] задать numLockOn=true и оно  будет работать сразу для всех пользователей, но только начиная с версии fly-wm >= 2.41.4. Узнать версию fly-wm можно командой:



fly-wm --version



В Astra Linux 1.6 без обновлений эта версия будет 2.17.29, а в Astra Linux 1.6 Update 10 как раз 2.41.4.
