Проблема: при каждом движении мыши или стрелок на клавиатуре в момент выделения текста, в буфер обмена копируется весь выделенный текст. Это сильно замедляет работу режима выделения.


Проблема решена в версии 1.2.0, комит 5dc8fe4.


Open micro config file /home/USER/.config/micro/settings.json and set next option:

"useprimary": true - for enable put symbols to PRIMARY buffer with each move of selection cursor

"useprimary": false - for put symbols to PRIMARY buffer only if push Ctrl+C.
