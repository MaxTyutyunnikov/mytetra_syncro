Проблема: при каждом движении мыши или стрелок на клавиатуре в момент выделения текста, в буфер обмена копируется весь выделенный текст. Это сильно замедляет работу режима выделения.


Проблема уже решена в версии 1.2.0, комит 5dc8fe4. Для отключения вышеписанного поведения, чтобы копирование в буфер обмена происходило только при нажатии Ctrl+C, можно сделать следующую настройку:


Открыть файл конфигурации /home/USER/.config/micro/settings.json и установить следующеи опции:


  • "useprimary": true - чтобы разрешить вставку символов в PRIMARY буфер обмена при кадлом движении курсора выделения
  • "useprimary": false - чтобы разрешить вставку символов в PRIMARY буфер только если нажато Ctrl+C.


Примечание: данная проблема актуальна только тогда, когда настроен проброс буфера обмена между сессей X11 и SSH с помощью ключа -X.
