Вроде как получить root права можно без установки постронних неизвестных программ.



Мнение alexkzru59 следующее:


Вообще не помню чтобы при получении рут прав удалялись пользовательские данные.

1. Извлекаем из прошивки boot.img
2. После чего устанавливаем Magisk
3. Кидаем boot.img на устройство
4. Патчим boot.img (пропатченый boot.img (имя будет другое) будет в папке Download), надо скопировать на ПК
5. Извлекаем из прошивки файл vbmeta.img копируем его туда же, куда boot.img
6. В папке с файлами вводим cmd или открываем PowerShell (для Windows)
7. Грузимся в Fastboot и выполняем команды:

fastboot flash boot boot.img

fastboot --disable-verity --disable-verification flash vbmeta vbmeta.img
