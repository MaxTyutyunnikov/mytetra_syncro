Данное решение проверено в Astra Linux 1.6, Astra Linux 1.6 upd 12, Astra Linux 1.7.


В менеджере файлов графической среды FLY имеется ветка Сеть. Если в домене поднят файловый сервер на базе Samba, то в этой ветке будут отображаться каталоги Samba. И в них даже можно зайти и управлять файлами. Проблема в том, что с таким подключением каталогов через протокол SMB, не все программы работают корректно. Например, LibreOffice может открыть файл, расположенный в сетевом файловом каталоге, если в менеджере файлов кликнуть по файлу. Но тот же LibreOffice выпадет в сегфолт, если попытаться открыть этот же файл стандарным способом в меню Файл - Открыть.
