Вид StackView используется для организации переключения страниц по принципу стека. Его имеет смысл использовать, когда интерфейс организован по правилу "страница режима - страница подрежима", то есть когда выход из подрежима происходит в предыдущий режим.


Чтобы переключиться на страницу, с помещением ее в стек, используется метод push(id_страницы). Чтобы выйти со страницы на предыдущую страницу, используется метод pop().

// Реализация возврата на предыдущую страницу при нажатии Key_Back на мобильном устройстве

// Срабатывает для всех страниц

Keys.onReleased: {

if (event.key === Qt.Key_Back)

pageBack(event);

}


// Реализация возврата на предыдущую страницу при нажатии ESC на обычной клавиатуре

// Срабатывает для всех страниц

Keys.onPressed: {

if (event.key === Qt.Key_Escape)

pageBack(event);

}


// Обработка нажатия кнопки выхода с текущей страницы

function pageBack(event) {

if( stackView.depth > 1 ) {

stackView.pop()

event.accepted = true

}

}
