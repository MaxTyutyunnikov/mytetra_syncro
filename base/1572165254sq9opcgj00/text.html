Если настройка яркости экрана в Linux не работает по горячим клавишам для ноутбуков с видеокарточкой Intel 915 и ей подобным (например для HP 450 ProBook), то яркость можно настроить просто редактируя определенный системный файл.


Перед редактированием надо узнать максимально возможную яркость, которую может выставить драйвер экрана. Для этого надо посмотреть содержимое следующего файла:



cat /sys/class/backlight/intel_backlight/max_brightness



Для разных моделей видеокарточек Intel, в этом файле хранятся разные максимальные значения яркости. Для встроенной видеокарточки Intel 915, скорее всего, будет прописано значение 3484.


Поменять значение яркости экрана можно через файл:



/sys/class/backlight/intel_backlight/brightness



Этот файл можно изменять только от суперпользователя (root). В нем нужно просто прописать число от нуля до максимально возможного значения ярктости. И, сразу после записи изменений в этом файле, яркость экрана установится на заданное значение. Не рекомендуется устанавливать самые нуливые значения, потому что будет сложно разглядеть что написано на экране, чтобы установить яркость, комфортную для зрения.
