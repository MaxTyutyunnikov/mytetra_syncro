В ffmpeg есть две опции:


-ss ЧЧ:ММ:СС - начать воспроизведение с указанного места

-t ЧЧ:ММ:СС - воспроизводить указанное количество времени


Например, чтобы получить кусок, начиная с 39-й минуты длительностью 10 сек, можно воспользоваться командой:


ffmpeg -i InputVideo.avi -f avi -vcodec copy -acodec copy -ss 0:39:00 -t 0:00:10 OutputVideo.avi
