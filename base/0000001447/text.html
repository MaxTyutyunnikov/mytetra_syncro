Чтобы прописать какую-то директорию в качестве репозитария, надо понимать, как apt преобразует ссылку.


Преобразование происходит следующим образом. Предположим, в /etc/apt/sources.list прописана такая строка:



deb http://repository.org/debian any main



Здесь any - это имя репозитария, а main - это имя секции. Обязательно нужно указывать и имя репозитария и имя секции (имен секций можетбыть несколько, разделяются так же пробелами). Иначе будет ошибка типа:



# apt-get update

E: Искажённая запись 3 в list из файла /etc/apt/sources.list (Component)

E: Не читается перечень источников.



Вышеприведенная запись означает, что будет производится попытка использовать в качестве репозитария директорию:


http://repository.org/debian/dists/any/main
