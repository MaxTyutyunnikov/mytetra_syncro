Сервис TRELLIS


Данный сервис представляет собой 3D-генератор на основе ИИ Trellis с открытым исходным кодом (за авторством Microsoft).


Некоторые энтузиасты запускают его, для демонстрации, в Docker-контейнере на специализированных хостингах:



https://huggingface.co/spaces/JeffreyXiang/TRELLIS



Этот хостинг предоставляет машины с подключенными видеокарточками,


Первые несколько генераций происходит бесплатно, и даже без регистрации. В России не работает, ошибка при загрузке изображения, необходимо заходить через VPN.


Картинку можно загружать в формате PNG с прозрачным фоном. Если загрузить в JPG вместе с фоном, то фон будет автоматически удален через утилиту (сравис?) rembg.


Сервис "додумывает" геометрию сзади предмета, додумывает "глубину" расположения элементов передней (видимой) части. И с использованием этих данных генерирует 3D-модель. Модель можно сохранить себе на компьютер.
