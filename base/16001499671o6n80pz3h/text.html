По сути, в 2020 году, в Рунете есть только один актуальный и живой форум по криптовалютам:



https://forum.bits.media/



Есть еще один, более-менее активный русскоязычный форум, в котором видно наличие ядра пользователей:



https://bitalk.org/



И никаких других вменяемых ресурсов в формате форума я в рунете больше не нашел. Все остальные ресурсы - это тематические группы ВКонтакте, например forklog, и сообщества внутри мессенджеров (обычно это Telegram). Называются группы типа Crypto Dealers, Cryptodealers News, CryptoEssay Chat и.т.д.
