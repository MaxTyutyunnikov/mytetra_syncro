Получить адрес функции в виде числа в языке Си++ можно через приведение указателя на функцию к типу void*. Делается это следующим образом:


// Код функции boo() находится в ячейке, например, памяти 002B1050

int boo()

{

return 7;

}

int main()

{

// Можно просто вывести указательна функцию,

// но его просто так нельзя запомнить, например, в unsigned int

std::cout << boo;


// Указываем C++ конвертировать функцию boo() в указатель типа void

std::cout << reinterpret_cast<void*>(boo);

return 0;

}


В консоль будет выведено:


002B1050

002B1050
