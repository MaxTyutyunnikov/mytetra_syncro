Имеется такой сайт:


http://htaccess.madewithlove.be/


Он представляет из себя онлайновый тестер фала .htaccess для веб-сервера Apache. Особенно необходим он в том случае, если нужно отладить правила модуля mod_rewrite.


Для тестирования надо ввести всего два поля:


Начальный URL

Содержимое .htaccess

HTTP_REFERER (если он необходим)


В результате будет отображен преобразованный URI и отладочная информация вот в таком виде:


Debugging info
