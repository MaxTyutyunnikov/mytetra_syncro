Центр вращения картинки задается внутри трансформации Rotation через свойства origin.x и origin.y. Это именно координаты на самой картинке, вокруг которых она будет вращаться.


Вот пример:


Image {

x: 1

y: 1

source: "arrow.png"

transform: Rotation {

origin.x: 9

origin.y: 67

angle: 50

}

}


Здесь картинка стрелки имеет размер 20x100, и выглядит так:



Если ее разместить в окне размером 100x100, и задать координаты (50, 50), то картинка разместится так:


Window {

visible: true

width: 100

height: 100


Image {

x: 50

y: 50

source: "pic/arrow.png"


}

}
