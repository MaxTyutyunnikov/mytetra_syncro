Центр вращения картинки задается внутри трансформации Rotation через свойства origin.x и origin.y. Это именно координаты на самой картинке, вокруг которых она будет вращаться.


Вот пример:


Image {

x: 96

y: 35

source: "arrow.png"

transform: Rotation {

origin.x: 9; origin.y: 67

angle: 50

}

}
