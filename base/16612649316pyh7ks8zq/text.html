По-умолчанию, сразу после установки Astra Linux 1.6 сервис ssh на 22-м порту не включен, но присутствует в системе. Чтобы проверить, что сервис действительно есть, можно воспользоваться командой (под рутом):



# systemctl status ssh



В ответ будет выведена информация о сервисе ssh. В строке "Active:" будет написано, запущен ли в данный момент этот сервис.


Для запуска сервиса можно ввести команду:



# systemctl start ssh



Для включения автостарта сервису ssh, необходимо воспользоваться командой:



# systemctl enable ssh



После этого действия можно перезагрузить компьютер, и ssh будет автоматически подниматься на 22-м порту.
