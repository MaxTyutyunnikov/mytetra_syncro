Заставить играть все ролики - не проблема, можно написать что-то вроде:


ls | grep -i avi | xargs -n1 mplayer


Если же надо пускать ролики на проигрывание по кругу, то это достигается опцией -loop 0 и маской * вместо xaкоторую нужно вставить в нужное место. Т.е. должно получиться где то так:


mplayer -loop 0 путь/*


или так


mplayer { путь/*} -loop 0
