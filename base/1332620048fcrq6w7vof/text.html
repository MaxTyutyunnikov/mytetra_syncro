На форуме в ветке про СТК-5000 выкладывалась ссылка на файл инструментов для СТК-5000. Для использования его с СТК-7000 нужно в нем сделать изменения. Открываешь этот файл текстовым редактором и сохраняя его структуру заменяешь параметры Control,SMB,Patch по таблице (приложение к руководству СТК-7000). Затем его сохраняешь как CTK7000.ins и пользуешься в Cakewalk(Sonar). Для band-in-box необходимо редактировать файл паттернов.Имеет расширение *.PAT - принцип такой же, но структура файла несколько иная. Да еще создается типа сноски на этот файл (там есть эти файлы для разных синтов). Самому мне это делать нет времени, да и проверять нужно на каждой конкретной модели синта.
