Числовая анимация


Пример анимации перемещения прямоугольника по оси X, для этого анимируется свойство x:


import QtQuick 2.6


Rectangle {

width: 300

height: 300

color: "lightblue"


Rectangle {

id: square

width: 100

height: 100

y: 150

color: "green"


NumberAnimation on x {

from:0

to: 150

duration: 3000 // Длительность анимации в миллисекундах

}

}

}


Здесь анимируется свойство x, но можно анимировать любое свойство, даже самостоятельно созданное через ключевое слово property. Имя анимирующегося свойства пишется после ключевого слова on.


У типа NumberAnimation как у наследника типа Animation есть свойство running.



В коде выше NumberAnimation прописано внутри объекта, свойство которого анимируется. Это потому, что после ключевого слова on должно писаться именно анимирующееся свойство. Не идентификатор со свойством, а только свойство. Поэтому такую анимацию невозможно написать вне объекта:


Rectangle {

id: square


width: 100

height: 100

y: 150

color: "green"

}


NumberAnimation on square.x {

from:0

to: 150

duration: 3000

}


- такой код работать не будет.


В результате получится зеленый квадрат, который перемещается слева направо:





Для NumberAnimation существует еще одна форма записи, без ключевого слова on:


NumberAnimation {

properties: "x"

from: 0

to: 100

duration: 3000

}


Такая форма записи в явном виде работать не будет. Она работает только в рамках переходов (Transition) или в рамках описания поведения (Behavior).
