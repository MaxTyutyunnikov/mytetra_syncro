Пример анимации перемещения прямоугольника по оси X:


import QtQuick 2.6


Rectangle {

width: 300

height: 300

color: "lightblue"


Rectangle {

id: square

width: 100

height: 100

y: 150

color: "green"


NumberAnimation on x {

from:0

to: 150

duration: 3000 // Длительность анимации в миллисекундах

}

}

}


Здесь анимируется свойство x, но можно анимировать любое свойство, даже самостоятельно созданное через ключевое слово property. Имя анимирующегося свойства пишется после ключевого слова on.


Видно, что NumberAnimation прописано внутри объекта, свойство которого анимируется. Это потому, что после ключевого слова on должно писаться именно анимирующееся свойство. Не идентификатор со свойством, а именно только свойство. Поэтому вне объекта невозможно написать так:


Rectangle {

id: square


width: 100

height: 100

y: 150

color: "green"

}


NumberAnimation on square.x {

from:0

to: 150

duration: 3000 // Длительность анимации в миллисекундах (здесь 3 сек.)

}


В результате получится зеленый квадрат, который перемещается слева направо:



NumberAnimation { properties: "x"; from: 0; to: 100; duration: 200 }
