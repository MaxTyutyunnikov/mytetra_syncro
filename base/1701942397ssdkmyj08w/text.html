Согласно официальной спецификации, Yamaha PSR E473 поддерживает формат, который называется Style File Format (SFF) compatible.


Имеется две версии формата, и E473 поддерживает только первую, которая называется SFF1.



Формат SFF1


Формат SFF1 представляет из себя обычный MIDI-файл Standard MIDI data (format 0), в котором в особом порядке прописаны все части стиля. Понять, что файл имеет формат SFF1 можно по его заголовку, обычно где-то в первых 100...256 байтах в HEX-представлении видна аббревиатура SFF1 (коды 53, 46, 46, 31).

Формат SFF2


Так же в природе имеется формат SFF GE (Guitar Edition). Он используется в "старших" моделях синтезаторов Yamaha, линейка S-, SX- или Genos/Tyros. Данный формат еще называют SFF2, поскольку первый маркер в этом MIDI-файле SFF2.


Формат SFF2 не поддерживается в E473. Но в сети есть бесплатная программа-конвертер от товарища Jørgen Sørensen, написанная на .NET, и работающая как под Windows, так и под Linux в WINE.


Style Format 2 Converter is a software program for converting YAMAHA keyboard format SFF2 style files to format SFF1 style files for older/smaller keyboards. SFF2 is also known as SFF GE.

To convert SFF1 format to various sub format styles use Style Old Format Converter.
