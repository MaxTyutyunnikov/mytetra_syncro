Найти книги и документацию по компилятора GCC можно на официальной странице проекта GCC:



https://gcc.gnu.org/onlinedocs



Вся документация на английском, перевода на другие языки нет.


По неизвестным причинам, документы, которые по ссылке называются, например GCC 10.2 Manual, на самом деле содержат документ Using the GNU Compiler Collection for gcc version 10.2.0. Наверно, так и задумано.


К данной записи прикреплен сверстанный PDF-файл пользовательской документации на GCC v.10.2.


Этот же документ доступен в HTML-форме по ссылке: https://gcc.gnu.org/onlinedocs/gcc-10.2.0/gcc
