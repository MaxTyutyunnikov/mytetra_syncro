Иногда при компиляции программы в C++ коде нужно учесть, под какую операционку компилируется программа.

У разных операционок разные требования к размещениям файлов исполнимых бинарников и файлов данных пользователя. Здесь рассказвается, как можно учитывать эти особенности. В качестве примера используется сборка под ОС MeeGo.

Основная идея - использовать внутри *.pro файла специальную переменную, которая будет указывать на то, под какую платформу идет сборка. Для MeeGo для этих целей можно использовать переменную с произвольным именем, например TARGET_OS_MEEGO.
