В DEB-based дистрибутивах имеется несколько способов получения полного списка пакетов, доступных в подключенном репозитарии.


Перед тем как получить список пакетов, надо обновить индекс пакетной базы через команду:



apt-get update



После чего можно генерировать список пакетов в репозитарии.



Вариант через dpkg


Получение списка пакетов выполняется следующей командой:



dpkg --list



Недостаток: по неизвестной причине этак команда покажет не все доступные пакеты. Почему некоторые пакеты не отображаются, выяснить не удалось.



Вариант через apt-cache


Выяснить, чем кеш пакетов отличается от индекса пакетов, из стандартных страниц документации невозможно. Поэтому откуда команда apt-cache берет данные не совсем ясно. В любом случае, вывод, который она генерирует может содержать больше пакетов, чем вывод команды dpkg. И это касается не только различного формата отображения, когда в разных строчках может показываться ghjcnj


Для вывода списка всех пакетов использвется команда:



apt-cache search '.*'
