2. Запрет сервиса NetworkManger


Далее необходимо отключить сервис network manager. Данный сервис обычно устанавливается вместе с графическими средами типа KDE/Gnome, но может стоять и отдельно. Перед ручным конфигурированием его нужно отключить:


Остановка сервиса и запрет его автостарта:


sudo systemctl stop NetworkManager

sudo systemctl disable NetworkManager


Для того чтобы далее снова включить сервис, можно будет воспользоваться командой:


sudo systemctl start network.service
