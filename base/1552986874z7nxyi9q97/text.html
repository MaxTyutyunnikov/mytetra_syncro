Иногда бывает так, что иконка Dropbox исчезает из трея. Обычно происходит это в какой-то момент после перезагрузки Linux. Точнее, иконка не исчезает из трея, а вместо иконки рисуется пустое место.


Как это исправить?


Есть фикс для этой проблемы. В автозапуске, вместо обычной команды запуска Dropbox, надо прописать следующую команду:



dbus-launch dropbox start -i



После этого изменения загрузка Dropbox всегда будет срабатывать корректно, и иконка будет всегда видна в систрее.
