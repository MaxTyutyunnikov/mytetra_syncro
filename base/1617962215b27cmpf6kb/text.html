В Windows 7 - Windows 10 имеется возможность назначить любой существующий каталог отдельным диском.


Например, если имеется каталог C:\DISK_D, то его можно "превратить" в диск, который будет виден в системе как обычный отдельный диск. Делается это следующей командой:



subst D: C:\DISK_D



Если при выполнении данной команды будет выдаваться ошибка:



Недопустимый параметр: D:



то это значит, что в системе уже присутствует диск с такой же буквой. Надо либо выбрать незанятую букву диска, либо поменять конфигурацию дисков, чтобы данная буква оказалась свободной.


Чтобы отключить этот "виртуальный" диск, надо воспользоваться опцией /d:



subst D: /d



В результате работы этой команды диск будет отключен.
