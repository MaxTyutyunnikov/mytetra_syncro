Здесь даны опции конвертирования исходного видео в формате 1440x1080 50FPS interlace.


Результирующее видео будет иметь прогрессивную развертку (т. к. применяется фильтр yadif)


Рабочие параметры конвертирования следующие:


ffmpeg -threads 2 \

-i input.MTS \

-vf yadif \

-vf scale=960:720:flags=bicubic \

-r 25 \

-vsync 1 \

-y \

-codec:v mjpeg \

-q:v 1 \

-g 0 \

-codec:a adpcm_ima_wav \

output.avi
