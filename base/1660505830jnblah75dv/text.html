В Интеренете есть сайт, с помощью которого можно проверить доступные порты своего компьютера, которые видны из сети Интеренет. По сути, это web-интерфейс, вызывающий программы проверки сети типа nmap, которые натравливаются на заданный IP-адрес.


Для начала надо узнать свой IP-адрес, например через сайт myip.ru.


Затем, для проверки открытых портов, можно перейти на сайт yougetsignal.com по следующей ссылке:


https://www.yougetsignal.com/tools/open-ports/


В результате будет открыт веб-интерфейс проверки портов, видимых на указанном хосте в сети Интернет.
