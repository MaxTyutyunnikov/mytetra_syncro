Открытие офисных файлов через файловый менеджер Midnight Commander происходит двумя методами. Первый метод - это определение типа файла чере


# Microsoft Word Document

regex/i/\.(do[ct]|wri)$

Open=/usr/lib/mc/ext.d/doc.sh open msdoc

View=%view{ascii} /usr/lib/mc/ext.d/doc.sh view msdoc

type/^Microsoft\ Word

Open=/usr/lib/mc/ext.d/doc.sh open msdoc

View=%view{ascii} /usr/lib/mc/ext.d/doc.sh view msdoc


...
