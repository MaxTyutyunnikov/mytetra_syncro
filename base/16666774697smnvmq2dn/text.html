В PostgreSQL нет специальных команд, генерирующих структуру таблицы на основе внутреннего представления таблицы. Это действие передано на откуп клиентским программам, которые могут на основе данных из системных таблиц построить структуру заданной таблицы. Консольный клиент psql этого функционала, естественно, не содержит.


Но в любом случа надо уметь хоть в какой-то форме увидеть, что из себя представляет структура таблицы, какие типы полей в ней используется. Для решения этой задачи можно воспользоваться следующей командой:



SELECT column_name, column_default, data_type

FROM INFORMATION_SCHEMA.COLUMNS

WHERE table_name = 'my_table';



Где my_table - это имя исследуемой таблицы.


Этот запрос возвращает название колонки, ее значение по умолчанию и тип данных. Если этого недостаточно, то можно экспериментировать со столбцами из таблицы INFORMATION_SCHEMA.COLUMNS и добавить то, что необходимо.
