Создание треков через браузер Google Crome в Android


Существует возможность создавать треки из KML-файлов на платформе Android без установки приложения "Мои карты". Для этого нужно в Android запустить браузер Google Chrome, и перейти на сайт:


https://maps.google.com


Появится карта примерно того района, откуда совершен выход в интернет.
