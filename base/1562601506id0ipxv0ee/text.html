Если в консоли linphonec в момент разговора нужно набрать DTMF-код, то это можно сделать следующим образом.


Для набора, например, кода "1", нужно в консоли так и написать "1" и нажать Enter:



linphonec> 1



Можно таким образом набирать и DTMF-последовательности, например для набора дополнительного номера. То есть, необходимо написать нужные цифры и нажать Enter. Другими словами, для набора DTMF-последовательности, в linphonec не существует отдельной команды. Просто если в строке команды присутсвуют только телефонные символы, в момент разговора они трактуются как необходимость тонового набра данных си
