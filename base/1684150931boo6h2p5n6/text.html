Есть такая программа - XiaoMiTool V2


Что такое XiaoMiTool V2 ?


  • XiaoMiTool V2 (XMT 2) - это неофициальный инструмент для вашего персонального компьютера, с помощью которого вы можете управлять ПЗУ и программным обеспечением вашего смартфона Xiaomi и легко вносить некоторые изменения.
  • XiaoMiTool V2 - это продолжение XiaoMiTool, с улучшенной стабильностью, большим количеством функций и улучшенной графикой.
  • XiaoMiTool V2 возьмет на себя большинство операций, которые вам потребуется выполнить, начиная с момента подключения устройства к ПК и заканчивая установкой нужного ПЗУ или программного обеспечения.
  • XiaoMiTool V2 может загрузить последние официальные ПЗУ, twrp, root с помощью magick и решить, каким способом лучше всего установить его на свое устройство
  • XiaoMiTool V2 - это универсальный инструмент, позволяющий упростить моддинг для всех, у кого есть смартфон Xiaomi.


Имеются сборки для операционных систем:


  • Windows 7/8/10 64/32 bit
  • Linux 64bit
  • Apple MacOS 64bit


Можно попробовать через не рутануть устройство.
