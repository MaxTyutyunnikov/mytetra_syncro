Иногда при переносе системы на другое железо стоит переустановить драйвера, отвечающие за графику в X-сервере. В Debian Linux это можно сделать командами:



sudo apt-get install --reinstall xserver-xorg-video-intel xserver-xorg-core

sudo dpkg-reconfigure xserver-xorg


Иногда такое действие помогает при возникновении ошибок загрузки X-сервера.


В данном случае показаны команды, переустанавливающие драйвера Intel HD. Для других видеокарточек имена пакетов с драйверами отличаются, но их легко можно найти в том же synaptic.
