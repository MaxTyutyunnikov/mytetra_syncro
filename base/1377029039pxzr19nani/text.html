На чистом Raspbin не работает звук из коробки. Для того, чтобы звук заработал, следует выполнить следующие команды:


sudo apt-get install alsa-utils

sudo modprobe snd_bcm2835


Эти команды запустят модуль ядра snd_bcm2835, отвечающий за вывод звука через данную разновидность аппаратного звукового кодека, используемого в Raspberry Pi. Загрузку данного модуля можно добавить в автозагрузку.


Возможно, сработает прописывание строки snd_bcm2835 в файле /etc/modules.


Если такое решение не сработает, надо попробовать прописать модуль в файл /etc/moprobe.d/alsa.conf. Формат этого файла весьма сложен и запутан, прописать можно по аналогии с другими звуковыми модулями, упоминаемыми в данном файле.
