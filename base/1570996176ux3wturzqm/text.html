Если стоит задача переустановки Windows 7/8/10, хорошей идеей будет предварительно сохранить лицензионные ключи Microsoft, используемые на компьютере.


Сделать это можно с помощью программ:


  • ABR (полное название - Activation Backup and Restore) http://directedge.us/content/abr-activation-backup-and-restore
  • Magical Jelly Bean Key Finder https://www.magicaljellybean.com/keyfinder/


Данные программы найдут все изветные им места, где хранятся ключи, и покажут ключи пользователю. Так же, имеется возможность сохранить ключи в простой текстовый файл. Сохранение в текстовый файл работает для бесплатных версий этих программ. Отличие от сохранения в специализированный файл состоит в том, что при сохранении в текстовый файл, восстановление и ввод ключей для ПО
