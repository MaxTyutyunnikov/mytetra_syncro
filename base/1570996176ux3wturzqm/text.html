Если стоит задача переустановки Windows 7/8/10, хорошей идеей будет предварительно сохранить лицензионные ключи Microsoft (и ключи прочих производителей ПО), используемые на компьютере.


Сделать это можно с помощью программ:


  • ABR (полное название - Activation Backup and Restore) http://directedge.us/content/abr-activation-backup-and-restore
  • Magical Jelly Bean Key Finder https://www.magicaljellybean.com/keyfinder/


Данные программы проверят все изветные им места, где хранятся ключи, и покажут ключи пользователю. Так же, имеется возможность сохранить ключи в простой текстовый файл. Сохранение в текстовый файл работает для бесплатных версий этих программ, а большего, по сути, и не требуется.


Полученный файл нужно будет сохранить на флешку или отправить самому себе по Email до момента переустановки Windows. В момент установки Windows или ПО Microsoft или ПО других призводитлелей, таких как Abobe или Borland, надо просто посмотреть в данном текстовом файле ключ от соответствующего ПО и ввести его в поле запроса лицензионного ключа.


Возможно, что потребуется интерактивная или телефонная активация лицензионного ключа. В любом случае, эта активация вполне законна, и производитель ПО обязан принять информацию о ключе и выдать ключ активации. Если для Windows использовался ключ OEM, то данный ключ должен работать при переустановке WIndow на тот же самый компьютер.
