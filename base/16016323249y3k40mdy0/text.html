Прекрасный учебник для средних школ, выпущенный в 1991 году.


В авторах можно увидеть Р.А. Свореня - это человек, который написал книги для детей "Электричество шаг за шагом" и "Электроника шаг за шагом".


Для выполнения кода из данной книги, можно воспользоваться системой КуМир (Комплект Учебных МИРов) - система программирования, предназначенная для поддержки начальных курсов информатики и программирования в средней и высшей школе. В системе КуМир используется школьный алгоритмический язык с русской лексикой и встроенными исполнителями Робот и Чертёжник.


Система Кумир разработана в ФГУ ФНЦ НИИСИ РАН по заказу Российской Академии Наук и распространяется свободно на условиях лицензии GNU 2.0. Скачивается с официального сайта по ссылке:



https://www.niisi.ru/kumir/dl.htm



Данная программа имеет версии под Windows, Linux и MacOs.
