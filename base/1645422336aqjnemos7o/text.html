В стандартном заголовке typeinfo библиотеки STD имеется инструмент, через который можно получать значения типов переменных или выражений в виде строки. Узнать наименование типа переменной можно следующим образом:



#include <iostream>

#include <typeinfo> // для typeid

int main()

{

short x(3);

short y(6);


std::cout << typeid(x).name() << " " << x << std::endl;


// Итоговый тип данных в выражении x + y

std::cout << typeid(x + y).name() << " " << x + y << std::endl;

return 0;

}



Результат в GCC будет:



s 3

i 9



А в MSVC результат будет другим:



short 3

int 9



То же самое справедливо и для типов, создаваемых пользователем:



#include <iostream>

#include <typeinfo>


struct Base { virtual ~Base() = default; };

struct Derived : Base {};

int main() {

Base b1;

Derived d1;

const Base *pb = &b1;

std::cout << typeid(*pb).name() << '\n';

pb = &d1;

std::cout << typeid(*pb).name() << '\n';

}



Результат в GCC:



4Base

7Derived



Результат в MSVC:



struct Base

struct Derived
